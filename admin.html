Админка каталога и главного слайдера

Редактирует каталог и слайды через GitHub API. После коммита GitHub Actions может автоматически заливать сайт на FTP-хостинг.

Подключение к GitHub

Для сохранения нужен Personal Access Token с правом записи в репозиторий. Пароль GitHub сюда не подходит.

Главный слайдер

Здесь редактируются заголовки и пути к изображениям для 4 слайдов на главной странице.

Загрузка изображений в GitHub

Сначала можно загрузить новые фото в репозиторий, потом вставить их пути в слайдер или карточку товара.

Каталог

Можно менять название, цену, категорию, основное фото, галерею и оба текста карточки/товара.

Сохранение

Кнопка ниже коммитит `catalog-data.js`, `data/homepage-content.json` и все загруженные изображения в выбранный репозиторий.
